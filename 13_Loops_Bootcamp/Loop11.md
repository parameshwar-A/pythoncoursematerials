# Pattern Printing - Big O

## Requirement
- To print a Big O based on the user Requirement of width

## Steps
- Get user input for the width of the Big O 
- Print the Big O with the user specified width

## Example:
Sample input\

Enter the width of the Big O: 5

Sample output\

```
 @@@
@   @
@   @
@   @
 @@@

```
